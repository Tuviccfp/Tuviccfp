## 👨‍💻 Sobre mim

Desenvolvedor **Full-Stack** com foco no ecossistema **JavaScript/TypeScript**, apaixonado por construir aplicações robustas, bem arquitetadas e com código limpo. Tenho interesse especial em backend — arquitetura de APIs, autenticação, boas práticas como **SOLID** e **Clean Code** — e gosto de explorar o frontend com interfaces modernas e performáticas.

Estou sempre buscando evoluir, seja estudando novos padrões de projeto, lendo livros técnicos ou construindo projetos do zero para colocar os conceitos em prática.

> *"Clean code is not written by following a set of rules. You know you are working on clean code when each routine you read turns out to be pretty much what you expected."* — Robert C. Martin
